This is an application which gives users NASA's APOD (Astronomy Picture of the Day)

After you clone this repo, you will need to go through the follwing steps:-
1. Get your own API Key from https://api.nasa.gov/ 
2. Make a .plist file in your Xcode Named as APIConfig 
3. Add a property named as API_KEY
4. And in the value field paste the API Key which you just received on mail from NASA
